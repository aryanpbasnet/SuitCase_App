SuitCase App

The SuitCase app is designed to help users efficiently manage their holiday shopping list. It simplifies the process of creating, editing, and organizing items needed for holidays. Below are the key functionalities of the SuitCase app.

Features
User Registration and Sign In

Secure User Accounts: Users can create accounts and sign in securely.

SQLite Database: User credentials are stored and validated using a local SQLite database.

Adding Items

Item Creation: Users can add items with details like name, price, description, and picture.

Image Picker Integration: Users can select item images using an external image picker library.

Editing Items

Modify Item Details: Users can edit item information by accessing the "Item Detail" screen and making changes.

Deleting Items

Swipe to Delete: Users can remove items by swiping left, with a confirmation prompt to prevent accidental deletions.

Swipe Decorator: Visual feedback using a red background during the swipe action.

Marking Items as Purchased

Swipe to Mark Purchased: Users can swipe right to mark items as purchased, with a green indicator.

Database Update: The item status is updated in the SQLite database.

Item Delegation

Sharing Items: Users can share item details via SMS to friends or family, enabling collaboration in the shopping process.
